# Dataset Card

Primary dataset: HAM10000 Skin Cancer MNIST.

Repository expectation:

- `data/archive/HAM10000_metadata.csv`
- `data/archive/HAM10000_images_part_1/`
- `data/archive/HAM10000_images_part_2/`
- optional segmentation directories with `segment` in the folder name

Source:

- Kaggle: `https://www.kaggle.com/datasets/kmader/skin-cancer-mnist-ham10000`
- Optional segmentations: `https://www.kaggle.com/datasets/tschandl/ham10000-lesion-segmentations`
